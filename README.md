# 🌡️ Temperature Prediction Using Machine Learning

## 📌 Project Overview
This project predicts temperature using Machine Learning techniques based on historical weather data. The objective is to build a predictive model that accurately estimates temperature by analyzing weather-related features.

## 🎯 Objectives
- Predict future temperature values.
- Perform data preprocessing and feature engineering.
- Train and evaluate Machine Learning models.
- Visualize trends and prediction results.

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📂 Dataset
The dataset contains historical weather information, including:
- Temperature
- Humidity
- Pressure
- Wind Speed
- Other weather-related features

## ⚙️ Machine Learning Workflow
1. Import the dataset
2. Data preprocessing
3. Handle missing values
4. Exploratory Data Analysis (EDA)
5. Feature selection
6. Train-test split
7. Model training
8. Model evaluation
9. Temperature prediction

## 📊 Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## 📁 Project Structure

## 🚀 How to Run
1. Clone the repository.
2. Install the required libraries:
   ```
   pip install -r requirements.txt
   ```
3. Open the Jupyter Notebook.
4. Run all cells to train the model and generate predictions.

## 📈 Results
The trained model predicts temperature using weather-related features and is evaluated using standard regression metrics.

## 🔮 Future Improvements

- Integrate real-time weather API data.
